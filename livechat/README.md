# 💬 LiveChat — настоящий мессенджер

Реальный мессенджер на Node.js + WebSocket. Работает через интернет между любыми людьми.

---

## 🚀 Деплой на Railway (бесплатно, 5 минут)

### 1. Создайте GitHub репозиторий
1. Зайдите на [github.com](https://github.com) → New repository
2. Назовите `livechat` → Create repository
3. Загрузите все файлы этой папки в репозиторий

### 2. Задеплойте на Railway
1. Зайдите на [railway.app](https://railway.app)
2. **Login with GitHub**
3. **New Project** → **Deploy from GitHub repo**
4. Выберите ваш репозиторий `livechat`
5. Railway автоматически определит Node.js и задеплоит

### 3. Получите URL
1. После деплоя нажмите на проект
2. Вкладка **Settings** → **Networking** → **Generate Domain**
3. Скопируйте URL вида `livechat-xxx.railway.app`

### 4. Используйте мессенджер
- Откройте `client/index.html` в браузере (или используйте URL Railway)
- В поле "Адрес сервера" введите: `wss://livechat-xxx.railway.app`
- Введите ник и войдите!
- Отправьте ссылку + адрес сервера другу

```
livechat/
├── server/
│   ├── index.js        ← WebSocket сервер
│   └── package.json
├── client/
```

---

## ⚙️ Локальный запуск

```bash
npm install
npm start
# Откройте http://localhost:3000
# В поле сервера введите: ws://localhost:3000
```

## 🔧 Возможности

- ✅ Регистрация по нику (без пароля)
- ✅ Поиск пользователей в реальном времени
- ✅ Личные сообщения
- ✅ Индикатор "печатает..."
- ✅ Статус онлайн/офлайн
- ✅ История сообщений (в памяти сервера)
- ✅ Работает между разными устройствами и странами

## ⚠️ Ограничения бесплатного Railway

- Сервер засыпает через 30 дней неактивности
- История сообщений сбрасывается при перезапуске сервера
- Для постоянного хранения нужна база данных (SQLite/PostgreSQL)
